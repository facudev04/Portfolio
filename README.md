# 🚀 Portfolio — Facundo Galeano

Portfolio personal desarrollado con React, TypeScript y Tailwind CSS. Incluye modo claro/oscuro, animaciones, formulario de contacto y diseño responsivo.

## ✨ Características

- Modo claro / oscuro con toggle en la navbar
- Typewriter animado en el hero
- Cursor glow que sigue el mouse
- Formulario de contacto con Web3Forms
- Diseño completamente responsivo
- Scroll suave entre secciones
- Sidebar con redes sociales (desktop)

```
src/
├── App.tsx        # Componente principal
├── index.css      # Estilos con @apply y variables de tema
└── main.tsx       # Entry point
```

## 🚀 Correr localmente

```bash
# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev
```

## 📦 Build y deploy

```bash
# Build de producción
npm run build

# Deploy a GitHub Pages
npm run deploy
```
